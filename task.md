# JobsAddis Implementation Task List

This is the task checklist of missing or partially implemented features identified in the Agreement Gap Analysis. We will proceed with implementing these step-by-step upon your instruction.

## 1. Job Seeker Features
- [ ] **A.5 Vacancy Alerts Subscription**: Create a subscription preference menu in the profile tab allowing seekers to sign up for notifications matching specific categories.
- [ ] **A.9 FAQ & Support Menu**: Add an FAQ page/view accessible from the app navigation containing common questions and support contact options.

## 2. Employer Features
- [ ] **B.4 Advertisement Package Selection**: Implement package selection cards, details, and duration options for posting jobs, with a mock checkout flow.
- [ ] **B.7 Employer Notifications**: Add real-time in-app alerts and Telegram notifications for employers when candidates apply or when job posts are near expiration.

## 3. Admin Features
- [ ] **C.2 Job Scheduling & Reposting**: Enable scheduling publication dates and reposting options inside the admin vacancy management tab.
- [ ] **C.3 Content Management**: Build an admin interface to manage system content, including onboarding welcome texts, vacancy templates, and FAQs.
- [ ] **C.4 Package & Expiration Management**: Create a dashboard tab to track, assign, and manage package durations and expirations.
- [ ] **C.5 Reporting & Analytics**: Add a reporting tab with charts showing vacancy performance, application stats, user growth, and package usage.
- [ ] **C.6 Notification Management (Broadcasts)**: Create a tool to broadcast system messages, warnings, or notifications to all users or specific roles.

## 4. Automation & Security Features
- [ ] **D.3 Automated Expiration Reminders**: Configure cron function or scheduling backend task for daily notifications about job expirations.
- [ ] **E.2 Centralized Activity Log**: Set up a database audit logging table and basic visualizer in the admin console to track changes.

- [ ] **F.3 Bot Configuration and Deployment files**
- [ ] **F.4 Technical documentation and setup guides**
